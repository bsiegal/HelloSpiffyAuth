This is a Spiffy UI application built just for you and running on your computer. This simple example shows you an easy form which makes a REST call to your server. You can edit it, build it, run it, share it with your friends, and build a product on top of it. The application is all yours.

Next steps

You've played with the project a little, now make some changes.

  1. Change this page - This page is a template with most of the code in HTML and the dynamic controls added later. Change the content in src/main/java/org/spiffyui/hellospiffyauth/client/MainPanel.html.
  2. Change the style - You can set any style in src/main/java/org/spiffyui/hellospiffyauth/public/HelloSpiffyAuth.css.
  3. Add a REST Bean - Make this REST call reusable by adding it to a REST Bean.
  4. Add a navigation bar - Easy navigation for your application is one of the great features of Spiffy UI.

After you make any change you can stop the server and run the Ant command to recompile your project. You can also use the Eclipse IDE to run this application in GWT dev mode.
